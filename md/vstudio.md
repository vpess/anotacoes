## Atalhos

- **F5**: Executa o código ou passa para o próximo breakpoint;

- **F9**: Cria um breakpoint na linha onde o cursor está;

- **F11**: Navega pelo código, a partir de um breakpoint;

- **Ctrl -**: Navega pelo código, mas no sentido reverso;

- **Ctrl KC**: Comenta uma linha de código;

- **Ctrl KU**: Desfaz o comentário da linha;

- **Ctrl KD**: Identa automaticamente o arquivo;

- **Ctrl .**: Renomeia uma variável ou método em todo o projeto.

## Funcionalidades

- **Breakpoint condicional**: Após criar um breakpoint, posicione o mouse sobre o breakpoint, e aparecerá o ícone de uma engrenagem. Clique na engrenagem, e será exibido um menu para criar a condição que acionará o breakpoint.

- **Extensão CodeMaid**: Permite limpar e simplificar códigos. Mais detalhes sobre o funcionamento [aqui](https://marketplace.visualstudio.com/items?itemName=SteveCadwallader.CodeMaid).
